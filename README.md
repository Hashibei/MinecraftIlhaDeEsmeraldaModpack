# Modpack da Ilha de Esmeralda

Esse é o Modpack da Ilha de Esmeralda Season 2 (1.21.1), o Modpack da Season 1 (1.18.2) não estará mais disponivel para download e será considerado depreciado.

## Como Instalar as coisas Necessárias para Jogar o Modpack

Você deve instalar:

1. Baixe e Instalar o [Java Adoptium 25](https://github.com/adoptium/temurin25-binaries/releases/download/jdk-25.0.3%2B9/OpenJDK25U-jdk_x64_windows_hotspot_25.0.3_9.msi).
2. Deinstale o TLauncher se você utiliza ele.
3. Baixe e Instale o [Freesm Launcher](https://github.com/FreesmTeam/FreesmLauncher/releases/download/2.2.1/FreesmLauncher-Windows-MSVC-Setup-2.2.1.exe).

## Como Baixar o Modpack

Você deve:

1. Clique aqui: [Clica em mim](https://github.com/Hashibei/MinecraftIlhaDeEsmeraldaModpack/releases/download/v-alpha-latest/IlhaDeEsmeralda-Client.mrpack).
2. Após me baixar, me arrasta para dentro do Freesm Launcher.
