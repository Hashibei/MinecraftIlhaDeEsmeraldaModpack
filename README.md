# Modpack da Ilha de Esmeralda

Esse é o Modpack da Ilha de Esmeralda Season 2 (1.21.1), o Modpack da Season 1 (1.18.2) não estará mais disponivel para download e será considerado depreciado.

## Como Instalar

Para instalar você deve:

1. Baixar e Instalar o [Java Adoptium 25](https://github.com/adoptium/temurin25-binaries/releases/download/jdk-25.0.3%2B9/OpenJDK25U-jdk_x64_windows_hotspot_25.0.3_9.msi).
2. Desinstalar o TLauncher se você utiliza ele.
3. Baixar e Instalar o [Freesm Launcher](https://github.com/FreesmTeam/FreesmLauncher/releases/download/2.2.1/FreesmLauncher-Windows-MSVC-Setup-2.2.1.exe).


## Como Baixar

Tem 3 formas de você baixar o Modpack:

1. Você pode baixar pelo [Releases do Modpack](https://github.com/Hashibei/MinecraftIlhaDeEsmeraldaModpack/releases/tag/modpacks).
2. Você pode baixar pelo [IlhaDeEsmeralda - Base](<modpacks/IlhaDeEsmeralda - Base 1.0.0.mrpack>)
